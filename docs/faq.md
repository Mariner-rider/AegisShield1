# FAQ

## Does AegisShield Cloud prevent every attack?
No. It is a defense-in-depth platform that reduces risk and improves response visibility.

## Can it run in monitor-only mode?
Yes. Monitor-only mode is supported for safer rollout and degraded enforcement scenarios.

## Are high-impact actions automatic?
No. High-impact actions require explicit approval.

## Can billing provider be swapped?
Yes. Billing uses a provider abstraction and can be replaced.

## Is threat intel auto-applied?
Only low-risk updates are auto-applied; higher-impact recommendations require review.

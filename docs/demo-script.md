   - Create org/project, choose integration, generate quickstart.
3. **Integration (2 min)**
   - Show SDK middleware/agent heartbeat.
4. **Detection (2 min)**
   - Trigger sample suspicious traffic and show detections.
5. **Response (2 min)**
   - Show policy-driven actions and approval-gated high-impact path.
6. **Billing/licensing (1 min)**
   - Show plan state and entitlement-aware behavior.
7. **Audit/ops (1 min)**
   - Show immutable records, observability, and docs.
8. **Limitations (1 min)**
   - Clearly state platform limitations and shared responsibility.

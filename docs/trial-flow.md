# Trial Flow

1. User signs up and creates organization.
2. First project/environment is created.
3. Trial credentials (API key / token) are issued.
4. Trial starts for 30 days.
5. Grace period applies on payment failure/expiry per policy.
6. On conversion, paid subscription state sync updates entitlements.
7. On non-conversion, platform may fall back to monitor-only based on plan policy.
